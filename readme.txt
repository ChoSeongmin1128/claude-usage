ClaudeUsage - macOS 메뉴바 Claude/Codex 사용량 모니터 (v1.2)

────────────────────────────────────
 설치 방법
────────────────────────────────────

1. ClaudeUsage.app을 원하는 위치로 이동합니다.
   (권장: /Applications)

2. 앱을 처음 실행하면 macOS에서 차단될 수 있습니다.
   - 시스템 설정 > 개인정보 보호 및 보안
   - 하단의 ClaudeUsage 차단 안내에서 "그래도 열기" 선택
   - 앱 재실행

3. 메뉴바 아이콘을 눌러 인증을 진행합니다.

────────────────────────────────────
 주요 기능
────────────────────────────────────

- Claude/Codex 사용량을 메뉴바에서 동시에 모니터링
- Popover에서 Claude/Codex 탭 전환 조회
- 세션키 우선 + OAuth 폴백 조회
- 공통 알림 설정 + Claude/Codex 개별 알림 ON/OFF
- 배터리/동심원/원형 등 메뉴바 아이콘 스타일

────────────────────────────────────
 인증 안내
────────────────────────────────────

- 세션키 경로는 429/Cloudflare/서버 오류 영향으로 간헐적으로 실패할 수 있습니다.
- 안정적인 조회를 위해 Claude CLI OAuth 인증을 권장합니다.
- 세션키가 정상일 때는 세션키 경로를 우선 시도합니다.
- Codex는 Codex 로그인 토큰(브라우저/CLI 연동 상태)을 사용합니다.

────────────────────────────────────
 알림 안내
────────────────────────────────────

- 공통 > 알림에서 임계값/기준(사용량/남은 사용량)을 설정합니다.
- Claude > 알림, Codex > 알림에서 서비스별 알림을 개별적으로 켜고 끌 수 있습니다.
- 전체 알림이 꺼져 있으면 서비스 알림도 모두 비활성화됩니다.

────────────────────────────────────
 업데이트 방법
────────────────────────────────────

1. 메뉴바 아이콘 클릭 > "vX.X 업데이트 가능" 배너에서 "다운로드"를 누릅니다.
2. 다운로드한 zip을 압축 해제합니다.
3. 실행 중인 기존 ClaudeUsage 앱을 완전히 종료합니다.
4. 새 ClaudeUsage.app으로 기존 앱을 교체(덮어쓰기)합니다.
5. 새 앱을 실행합니다.
   - 최초 실행 시 차단되면:
     시스템 설정 > 개인정보 보호 및 보안 > "그래도 열기"

────────────────────────────────────
 문제 해결
────────────────────────────────────

- 조회 실패가 반복되면 잠시 후 자동 재시도됩니다.
- 세션키 기반 조회가 불안정하면 OAuth 인증을 먼저 구성해 주세요.
- 메뉴바 아이콘이 숨김 영역으로 이동하면(예: Ice) 메뉴바 관리 앱 설정에서 고정/표시 상태를 확인해 주세요.
