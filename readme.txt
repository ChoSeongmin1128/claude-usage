ClaudeUsage - macOS 메뉴바 Claude 사용량 모니터

────────────────────────────────────
 설치 방법
────────────────────────────────────

1. ClaudeUsage.app을 원하는 위치로 이동합니다.
   (예: /Applications 또는 바탕화면)

2. 앱을 처음 실행하면 "앱을 열 수 없습니다" 메시지가 나타날 수 있습니다.
   아래 방법으로 허용해주세요:

   - 시스템 설정 → 개인정보 보호 및 보안 으로 이동
   - 아래쪽으로 스크롤하면 "ClaudeUsage.app이(가) 차단되었습니다" 메시지가 표시됩니다
   - "그래도 열기" 버튼을 클릭
   - 이후 앱을 다시 실행하면 정상적으로 열립니다

   * 이 과정은 최초 1회만 필요합니다.

3. 메뉴바에 아이콘이 나타나면 클릭하여 Claude 로그인을 진행합니다.

────────────────────────────────────
 업데이트 방법
────────────────────────────────────

새로운 버전이 나오면 앱 내부에서 알려드립니다.

1. 메뉴바 아이콘 클릭 → "v X.X 업데이트 가능" 배너가 표시됩니다.
2. "다운로드" 버튼을 클릭하면 브라우저에서 새 버전 zip 파일이 다운로드됩니다.
3. 다운로드된 zip을 압축 해제한 뒤 기존 앱을 교체합니다.
4. 새 버전을 처음 실행할 때 위 설치 방법과 동일하게 보안 허용이 필요할 수 있습니다.
