==========================================
  ClaudeUsage 설치 방법
==========================================

이 앱은 Apple Developer 인증서로 서명되지 않은 개인 배포 앱입니다.
macOS는 미서명 앱을 기본적으로 차단하므로, 아래 방법으로 설치해주세요.

[방법 1] 시스템 설정에서 허용 (일반 사용자 권장)
-------------------------------------------
1. ClaudeUsage.app을 더블클릭합니다 (차단 경고가 뜹니다)
2. 시스템 설정 → 개인정보 보호 및 보안 으로 이동합니다
3. 아래로 스크롤하면 보안 항목에
   "Mac을 보호하기 위해 'ClaudeUsage'을(를) 차단했습니다" 메시지가 보입니다
4. "그래도 열기" 버튼을 클릭합니다
5. 이후 정상적으로 실행됩니다

[방법 2] 터미널에서 설치
-------------------------------------------
1. 터미널 앱을 엽니다 (Spotlight에서 "터미널" 검색)
2. 아래 두 명령어를 순서대로 실행합니다:

   sudo xattr -rd com.apple.quarantine ~/Downloads/ClaudeUsage/ClaudeUsage.app
   cp -R ~/Downloads/ClaudeUsage/ClaudeUsage.app /Applications/

   ※ 압축 해제 경로가 다르면 해당 경로로 변경해주세요.

3. /Applications 폴더에서 ClaudeUsage를 실행합니다.
