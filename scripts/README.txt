==========================================
  ClaudeUsage 설치 및 사용 가이드
==========================================

Claude 사용량을 macOS 상단 메뉴바에서 실시간으로 확인할 수 있는 앱입니다.


[1단계] 앱 설치
==========================================

이 앱은 Apple Developer 인증서로 서명되지 않은 개인 배포 앱입니다.
처음 실행 시 macOS가 차단할 수 있으며, 아래 방법으로 해결할 수 있습니다.

(방법 A) 시스템 설정에서 허용 — 일반 사용자 권장
-------------------------------------------
1. ClaudeUsage.app을 /Applications 폴더로 드래그하여 복사합니다.
   (Finder에서 ClaudeUsage.app을 드래그 → 왼쪽 사이드바의 "응용 프로그램"에 놓기)
2. /Applications 폴더에서 ClaudeUsage.app을 더블클릭합니다.
   → "열 수 없음" 차단 경고가 뜹니다. 이것은 정상입니다.
3. 왼쪽 상단 Apple 메뉴() → 시스템 설정 → 개인정보 보호 및 보안 으로 이동합니다.
4. 아래로 스크롤하면 보안 항목에
   "Mac을 보호하기 위해 'ClaudeUsage'을(를) 차단했습니다" 메시지가 보입니다.
5. "그래도 열기" 버튼을 클릭합니다.
6. 이후부터는 정상적으로 실행됩니다.

(방법 B) 터미널에서 설치
-------------------------------------------
1. 터미널 앱을 엽니다.
   (Spotlight 검색: Cmd + Space → "터미널" 입력 → Enter)
2. 아래 두 명령어를 순서대로 복사하여 붙여넣고 실행합니다:

   sudo xattr -rd com.apple.quarantine ~/Downloads/ClaudeUsage/ClaudeUsage.app
   cp -R ~/Downloads/ClaudeUsage/ClaudeUsage.app /Applications/

   ※ sudo 비밀번호를 입력해주세요.
   ※ 압축 해제 경로가 ~/Downloads/ClaudeUsage가 아니면 해당 경로로 변경해주세요.

3. /Applications 폴더에서 ClaudeUsage를 실행합니다.


[2단계] 세션 키 설정
==========================================

앱을 처음 실행하면 설정 창이 나타납니다.
Claude 사용량을 가져오려면 세션 키가 필요합니다.

세션 키 가져오는 방법:
-------------------------------------------
1. 웹 브라우저에서 https://claude.ai 에 로그인합니다.
2. 개발자 도구를 엽니다 (Cmd + Option + I).
3. Application 탭 → 왼쪽 메뉴에서 Cookies → https://claude.ai 를 클릭합니다.
4. 목록에서 "sessionKey" 항목을 찾아 값(Value)을 복사합니다.
   (sk-ant-sid01-... 으로 시작하는 긴 문자열)
5. ClaudeUsage 설정 창의 세션 키 입력란에 붙여넣고 저장합니다.

※ 세션 키는 로컬에만 저장되며 외부로 전송되지 않습니다.
※ Claude.ai에서 로그아웃하면 세션 키가 만료되므로 다시 설정해야 합니다.
